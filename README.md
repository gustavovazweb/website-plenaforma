# Centro Plena Forma

## Description

This is just a simple website for a gym.

## Technologies:
* Backend: PHP 7.
* Frontend: HTML, CSS and Javascript.

## Getting Started
* XAMPP (PHP, Apache, MySQL)
* Web Browser (Chrome, Firefox, Opera...)
* Code Editor (Visual Studio Code, Sublime...)
